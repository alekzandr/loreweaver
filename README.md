A generative D&D exploration encounter tool with a slick web interface.

## Overview

LoreWeaver generates complete exploration encounters for Dungeons & Dragons, featuring skill challenges, traps, hazards, and environmental effects across diverse environments.

## Features

- **10 Unique Environments**: Urban, Arctic, Ocean, Space, Crypt, Forest, Desert, Mountain, Swamp, and Underground
- **Skill Challenges**: Appropriate skill checks for each environment with dynamic DC scaling
- **Traps & Hazards**: Environment-specific dangers with detection and disarm mechanics
- **Environmental Effects**: Persistent conditions that affect the entire encounter area
- **Level Scaling**: Automatic adjustment of DCs and damage based on party level (1-20)
- **Slick UI**: Modern, responsive design with smooth animations and intuitive controls

## How to Use

1. Open `index.html` in any modern web browser
2. Select an environment by clicking on one of the environment tags
3. Set your party's level (1-20)
4. Click "ğŸ² Generate Encounter" to create a unique exploration encounter
5. Each generated encounter includes:
   - Encounter title and description
   - 3-4 skill challenges with appropriate DCs
   - 1-3 traps with detection/disarm mechanics
   - 2-3 environmental hazards
   - 1-2 persistent environmental effects
   - Level-specific scaling guidance
   - Resolution and reward suggestions
